## Ⅰ. サッカーの4局面（すべての練習の基盤）

| 局面 | 状態 | 指導の焦点 |
|------|------|-----------|
| **攻撃（ポゼッション）** | ボールを保持している | 前進・崩し・フィニッシュ |
| **ネガトラ（守備への切り替え）** | 失った直後の0〜3秒 | 即時奪回 or 守備組織形成 |
| **守備（非ポゼッション）** | 相手がボールを保持 | ゴール保護・ボール奪取 |
| **ポジトラ（攻撃への切り替え）** | 奪った直後の0〜3秒 | 即前進・数的優位の活用 |

**ポジトラの原則 = 奪った瞬間に最優先で前進（深さを作る→ライン間→保持）**

## Ⅱ. 技術の本質（風間八宏哲学）

> 「技術が完成すれば時間が生まれ、判断の選択肢が増える」

### 止める（トラップの本質）
- **定義**：自分がどこへでも蹴られるポイントにボールを完全静止させること
- 「転がっている・バウンドしている = 止まっていない」
- 止めた瞬間からの次の動作（蹴る・運ぶ）を0.5秒以内
- **止めることで「時間が生まれる」** = 判断の余裕

### 蹴る（キックの本質）
- **定義**：ボールに正確に力を伝えること
- インパクトポイント（足のどこで当てるか）が精度を決める
- 蹴る目的（パス・シュート・クリア）に応じて力の方向・強さを選択

### 受ける（受け方の本質）
- ボールを受ける前に「次に何をするか」の判断が完了している
- **首を振る習慣**：ボールを受ける前に周囲の状況を確認
- 体の向きが次のプレーを決定する

### 外す（マーカーを外す本質）
- 「止める・蹴る」ができることで外す（マーカーを動かす）が成立する
- フェイント・体の向き・ボールの置き場所で相手を外す
- 技術が高いほど外す選択肢が増える

## Ⅲ. ドリブル理論（岡部将和 ドリブルデザイン）

> 「ロジック（原理）× テクニック（技術）= 抜けるドリブル」

### ドリブルの2要素
- **ロジック**：なぜ抜けるか の理論（相手の重心・角度・間合いの理解）
- **テクニック**：それを実行するスキル

### 間合いの原則
- **適切な間合い** = 相手の足が届かず、かつ相手の体でもブロックできない距離
- 近すぎると相手にカットされる、遠すぎると仕掛けの効果がない
- 相手のスキル・スピード・利き足によって間合いを変える

### 重心操作の原則
- **相手の重心を読む**：相手がどちらの足に重心をかけているかを見る
- 重心の逆をつく = 「後出しじゃんけん」で勝てる
- 相手の重心を後ろに動かしてからカットイン（縦→中の順）
- 縦への仕掛けで相手の重心を引き出し、切り返しで逆をつく

### ドリブルの種類と使い分け
- **縦ドリブル**：スペースがある時に加速で一気に抜く
- **カットイン**：縦に見せて内側へ切り込む（重心が縦方向の時に有効）
- **フェイント**：体の向き・ボールタッチで相手を動かす

## Ⅳ. 攻撃の判断フレームワーク（ポジショナルプレー × 日本の指導実践）

### 攻撃の判断優先順位

| 優先順位 | 名称 | 説明 |
|---------|------|------|
| **① 縦への前進（突破）** | 相手の背後・深さ | DFラインの裏へのパス/ランで最も危険な場所を狙う |
| **② ライン間受け（サポート）** | 相手ラインとラインの間 | 相手の守備ラインとラインの間でボールを受けて前進 |
| **③ 保持・リセット（循環）** | 横・後ろへのパス | 安全を確保しながらやり直し、別のコースを探す |

### 体の向きの原則（正対）
- **正対** = 最も近い相手に体正面を向けてボールを持つ
- 正対することで複数の選択肢を同時に持てる（右にも左にも出せる）
- 相手は次のプレーを予測できなくなる
- 逆に体が「向いていない」状態でのパスは相手に読まれやすい

### ポジトラ（ボールを奪った直後）の判断
1. 縦への前進が可能 → 即前進（深さを作る）
2. 縦が難しい → ライン間のサポートを探す
3. サポートもない → 保持してやり直す

## Ⅴ. ポジショナルプレー（スペイン流）

> 「ボールの場所に合わせて攻守にかかわらず選手が最適なポジションをとり優位性を確保する概念」

### 5レーン理論の基本

```
サイド │ハーフ│ センター │ハーフ│ サイド
レーン │スペース│  レーン  │スペース│ レーン
```

**ハーフスペース**（サイドとセンターの間）が最も崩しやすいゾーン。
相手がサイドを閉じてもセンターを閉じても、ハーフスペースが空く。

### 5レーンの3原則
1. **各レーンに最大2人**：被らず、スペースを有効活用
2. **各横ゾーンに最大3人**：コンパクトかつ展開が可能な配置
3. **隣接レーンの選手は斜めの位置関係**：三角形を作り、パスコースを確保

### 優位性の3種類
- **数的優位**：相手より人数が多い状況
- **位置的優位**：相手より良いポジションにいる（前向き・フリー）
- **質的優位**：1v1で勝てる技術的な差

### ビルドアップ時の配置原則
- GKを含めた後方での数的優位 → 前線への進行ルートを作る
- CBは幅を広げてプレスを外す
- ボランチは「ライン間」でボールを受けられる位置に
- WGは幅＋深さを確保（相手SBを引き出す）
- FWは「深さ」を作り相手DFラインを下げる

## Ⅵ. 守備の原則

### 守備の4段階（優先順位）
1. **インターセプト**：パスコースを読んでボールをカット（最も効率的）
2. **チャージ**：相手がコントロールした瞬間を狙って激しく寄せる
3. **ディレイ**：すぐに取れない場合は時間を稼いで味方のカバーを待つ
4. **クリア**：最後の手段。ゴール前からボールを排除

### 1v1守備の原則（岡部理論の逆用）
- 相手の「間合い」に入らせない → 相手の得意な間合いを崩す
- 相手の重心移動を見てから動く（先に動かない）
- 両足をそろえない、重心は体の中央
- 抜かれる方向を「縦」と決めて守る（最悪の場面を避ける）
- カバーが来るまで「時間を稼ぐ」ことが守備の貢献

### グループ守備の原則
- **ハイプレス**：相手陣内または中盤でボールを奪いに行く組織的プレス
- **ミドルブロック**：自陣中盤に守備ブロックを作る
- **ローブロック**：自陣深くに引いてゴールを守る
- **コンパクト**：ラインとラインの間を狭く保ち、スペースを消す

### ネガトラ（守備への切り替え）の原則
- **失った瞬間**：6秒以内に即時奪回を狙う（ゲーゲンプレス）
- 無理なら素早く守備組織を作る（同サイドに圧縮）
- DFラインを上げて全体をコンパクトに保つ

## Ⅶ. 各ポジションの基本原則

### GK（ゴールキーパー）
- ビルドアップ時：フィールドプレーヤーとしてボールを受ける
- セービング：ゴールとボールと体の「3点」の関係を保つ
- 飛び出しの判断：確実に取れる場面のみ

### CB（センターバック）
- 幅を広げてビルドアップのパスコースを作る
- 縦への持ち運び（ドリブル）で相手を引き出す
- 空中戦・1v1の強さが基本条件

### SB（サイドバック）
- 攻撃時：オーバーラップ or インナーラップで幅と深さを提供
- 守備時：相手WGへの対応 + 中への絞り
- **最大限の高さ**（前向きでボールを受けられる限界まで高い位置）を保つ

### ボランチ（中盤MF）
- 攻撃：「ライン間」でボールを受け、縦に出す
- 守備：中盤のフィルター（縦パスコースを消す）
- ポジトラ：ボール奪取の起点として即座に前進

### ウインガー
- 幅をとって相手SBを引き出す → ハーフスペースを開ける
- 1v1での突破 or カットインからのシュート
- 守備時：自陣への素早いプレスバック

### FW（トップ）
- **深さを作る**（DFラインを下げる）ランニング
- 裏抜け：相手DFより先にスペースに入る（「のり代」をつけて走る）
- 前線からの守備でプレスのスイッチを入れる

## Ⅸ. ドリルの論理連鎖 — セッション設計の鉄則

各ドリルは「出力スキル」を次のドリルの「入力スキル」として渡す。

```
[A. 身体準備（ウォームアップ）]
  目的：その日のテーマに必要な「体の動き」を準備
  例：横への重心移動 → ドリブルの切り返し準備
  例：首振り・視野確認 → パス判断の準備

[B. 技術習得（テクニカルトレーニング）]
  目的：テーマのコアスキルを反復で身体に入れる
  例：止める・蹴るの精度練習 → パスの質向上
  例：間合い・フェイント練習 → 1v1の準備

[C. 個人判断（1v1・小グループ）]
  目的：習得したスキルを「判断」と組み合わせる
  例：1v1（守備との駆け引き）
  例：2v1（数的優位の活用）

[D. グループ戦術]
  目的：複数人でのコンビネーション・戦術の体験
  例：4v2ポゼッション → 優先順位の判断
  例：崩しのパターン練習

[E. ゲーム（SSG〜フルゲーム）]
  目的：すべてを実戦に近い状況で発揮する
  オーガナイズ：今日のテーマが「自然に出る」ルール設定が必須
```

**セッション設計の鉄則：A → B → C → D → E の順番、かつ各ドリル間に「なぜ繋がるか」を説明できること**

## Ⅹ. コーチングのアプローチ原則

### 技術指導（風間流）
- ○× をつけられる明確な基準を持つ（「止まった / 止まっていない」）
- 「感覚」でなく「見える化」できる技術指導を心がける
- 年齢関係なく同じ基準で技術を評価する

### ドリブル指導（岡部流）
- 「なぜ抜けるか」の原理から教える（結果でなく原因）
- 相手の重心・間合いを「観察する力」を育てる
- 失敗したときの「なぜ抜けなかったか」を一緒に分析する

### グループ指導（ポジショナル流）
- ポジションのずれ・被りに気づかせる（5レーンの感覚）
- 「優位性を作れているか」を常に問いかける
- スペースを見る目（スキャニング習慣）を育てる

### 全体指導
- 「どうすれば良かったか」を選手自身に言わせる（主体性）
- コーチングポイントは1回の練習で1〜2個に絞る
- 試合に近い状況（プレッシャー下）で技術を確認する
