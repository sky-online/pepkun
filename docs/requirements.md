# ペップ君 — 要件定義書

> 最終更新：2026-05-28（山村さんのビジョンヒアリング反映）

---

## プロダクトビジョン

**「Jクラブの指導知見を、すべてのチームへ。」**

日本には選手はいるが、質の高いコーチングを受けられていない子どもたちが多い。
ボランティアコーチ・OBコーチ・学生アルバイトコーチでも、
Jクラブ・一流指導者レベルのトレーニングが実現できる環境を作る。

---

## 解決する課題

| 課題 | 現状 | ペップ君が解決すること |
|------|------|---------------------|
| 指導者の知識不足 | ボランティア・OBが多く、体系的に学べていない | 最新のサッカー理論に基づくメニューを自動生成 |
| メニューの非論理性 | 各練習がバラバラで繋がっていない | ウォームアップ〜ゲームまで一気通貫の論理構成 |
| チームコンセプトの未反映 | 練習が個人技術の羅列になりがち | チームスタイル・コンセプトを起点にメニューを設計 |
| PDCAが回っていない | 練習後の振り返りがない・次に活かせない | CIレポート → 改善抽出 → 次月計画の自動化 |

## システム構成（5モジュール）

### MODULE 1: チームプロファイル
```
- チーム名・カテゴリ（U-8〜成人）
- 人数・練習頻度・使える環境
- チームコンセプト（ポゼッション / カウンター / プレッシング etc.）
- プレースタイル・大事にしている原則
- 蓄積されたCI履歴
```

### MODULE 2: サッカー原理原則ナレッジベース（最重要）
```
サッカーの本質的な原則を定義したコア知識。
すべてのメニュー生成の「根拠」になる。

例：
- 攻撃の原則：前進・幅・深さ・サポート
- 守備の原則：ゴール前に壁を作る・コンパクト・プレッシャー
- ビルドアップ原則：数的優位・ポジショニング
- 年代別原則：U-12はスキル習得優先、U-15以降は戦術理解
```

### MODULE 3: セッション生成AIエンジン（コア）
```
入力：チームプロファイル + セッションテーマ + 制約条件
出力：論理的に繋がった完全セッション

【論理連鎖の設計】
ウォームアップ
  ↓（動きのパターンを植える）
テクニカルトレーニング
  ↓（スキルを単独で習得）
小グループ戦術
  ↓（スキルを状況判断と組み合わせる）
ゲーム形式
  ↓（すべてをゲームで発揮）
クールダウン＋振り返り

例：
ラダーのステップワーク
→ ドリブルで相手を剥がすドリル
→ 1v1 / 2v2でその剥がしを使う
→ ゲーム（剥がしが活きるオーガナイズ設計）
```

### MODULE 4: CIエンジン（Continuous Improvement）
```
入力：練習後の指導者レポート（音声 or テキスト）
処理：
  - 何が良かったか / 悪かったかを抽出
  - チームの現在課題を特定
  - 原理原則ベースで原因を分析
出力：
  - 改善ポイント（具体的）
  - 次回セッションへの反映事項
  - チームの成長トラッキング
```

### MODULE 5: 練習プランビルダー（PDCA）
```
入力：蓄積されたCIデータ + チームゴール
出力：1ヶ月の練習計画
  - 週別テーマ設定
  - テーマの論理的順序（積み上げ型）
  - 試合日程との連動
```

---

## 入力パラメータ（セッション生成）

```
必須：
- 対象年齢：U-8 / U-10 / U-12 / U-15 / 成人
- 人数
- 練習時間（分）
- 今日のテーマ（例：「ビルドアップ」「プレッシング」「ドリブル突破」）

チームコンテキスト（プロファイルから自動引用）：
- チームコンセプト
- 直近のCI課題
- 月間プランでのこの日の位置づけ

オプション：
- 使える器具
- 場所・スペース
- 特記事項（例：「今日は怪我人が多い」「試合前日」）
```

## 出力フォーマット（生成メニュー）

```
【ペップ君 練習メニュー】
━━━━━━━━━━━━━━━━━━━━━━━━
チーム：〇〇FC　U-12 / 20人 / 90分
テーマ：ビルドアップ〜前進
コンセプト接続：ポゼッションから深さを作る
━━━━━━━━━━━━━━━━━━━━━━━━

■ ウォームアップ（15分）
  [ラダー：ラテラルステップ系]
  目的：横方向の重心移動を活性化
  　→ これはこの後のボール受ける前の動き出しに直結します

■ テクニカル（20分）
  [2タッチパス＋ターン]
  目的：受ける前の首振りとタッチ後の加速
  　→ ビルドアップの「前を向く」原則を身体に入れる

■ グループ戦術（25分）
  [4v2 ポゼッション → 縦パス優先]
  目的：ポゼッションしながら前への意識を持つ
  　→ テクニカルで習得した「前を向く」を判断に組み込む

■ ゲーム（25分）
  [4v4 + フリーマン ビルドアップゲーム]
  組織：GKからビルドアップを必ず繋ぐルール
  目的：今日学んだすべてをゲームで発揮

■ クールダウン＋振り返り（5分）
  今日のポイント確認：「前を向く前に何をした？」

━━━━━━━━━━━━━━━━━━━━━━━━
【コーチングポイント】
・ボールを受ける前の「準備」を常に声かけ
・ミスを咎めず「なぜそうなった？」を問いかける
・ラダーとゲームの動きが繋がっていることを選手に伝える
━━━━━━━━━━━━━━━━━━━━━━━━
```

## 品質基準

- JFA指導指針との整合性
- 年代発達段階（認知・身体・技術）への適合
- 練習内の**論理的連鎖**が必ず説明されていること
- 「なぜこの練習順か」がコーチに分かること
- 実際に現場で使えるレベルの具体性
