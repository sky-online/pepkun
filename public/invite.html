How it works

01
チームのことを話す
チーム名・年代・人数・スタイルをAIコーチに伝える。一度話せば次回からは覚えている。毎回入力し直す必要はない。
02
今日のテーマを伝える
「前進のパス回し」「切り替えの速さ」「サイド攻略」——テーマを一言伝えるだけで生成が始まる。迷ったらAIが提案する。
03
メニューを受け取る
5つのドリル・配置図・声かけ例・トラブル対処が揃った練習メニューが数秒で届く。そのまま現場で使える。

Sample

生成される練習メニュー

ゴール前の決定力を高める！シンプル＆正確フィニッシュセッション
U-12 12名 60分 シュート・得点力
01
Warmup
ゴールハンター・ボールマスタリー＋シュートゲート
ペナルティエリア付近でコーン8個のゲートを散在配置。合図でボールを最寄りのゲートに転がして通過させる。インサイド・アウトサイド切り替えを意識しながらゴールを狙う感覚をウォームアップから作る。
10分 / 20m×20mグリッド / コーン16本・ボール12個
02
Technical
3ポイント・インサイドシュート精度ドリル
ゴール正面5m・斜め左7m・斜め右7mの3カ所にシュートポイントを設置。「止める→軸足をボールの横に置く→インサイドで振り抜く」の3ステップを声に出しながら繰り返す。ポイントをローテーション。
12分 / 3列4名 / ゴール1個・コーン6本
03
Technical
ファーストタッチ→前向きシュートコンビネーション
2列（各6名）でサイドからパスを受け、ファーストタッチで前を向いてシュートを打つ。「ボールを迎えに行く→体を横向きにして受ける→ゴールに向かってタッチ→シュート」の流れを2タッチ以内で完結させる。
13分 / 2列6名 / ゴール1個・ボール4個
04
Tactical
3人連動フィニッシュ・ポジショナルパターン
CB→MF（ターン）→FW（斜め抜け出し）→シュートの3人連動パターンを繰り返す。慣れたら「MFがシュート」「FWが落としてMFがシュート」のバリエーションも追加し、動き出しのタイミングと判断を磨く。
15分 / 3人組×4組 / ハーフコート・ボール4個
05
Game
シュートゾーン侵入ゲーム 4v4+GK
ペナルティエリア前を「シュートゾーン」として設定。ゾーン内からシュートして決めたゴールは2点・ゾーン外は0点。練習してきたシュートフォーム・ファーストタッチ・連動パターンをゲームの中で自然に発揮させる。
10分 / 4v4+GK×2組 / ハーフコート・コーン4本

Features

チームを記憶する
一度伝えたチーム情報はAIが記憶する。毎回入力し直す必要はない。セッションを重ねるごとに提案の精度が上がる。
配置図つきで届く
各ドリルにフィールド配置図が自動生成される。選手・コーン・動きの矢印が視覚化され、現場でそのまま使える。
声かけ例まで入っている
コーチングポイントだけでなく、実際に口に出す言葉と練習を止めるタイミングまで設計されている。
名将の知識をベースに設計
ペップ・グアルディオラ、風間八宏、クロップ、オシム——名将たちの指導哲学をAIが学習し、理論的な裏付けのあるメニューを生成する。
年代・レベルに合わせて変わる
U-8からシニアまで、年代別発達段階に応じた言語と複雑度でメニューを設計する。難しすぎる・簡単すぎるが起きない。
